# DA_brighttv_project
OVERVIEW
This project analyzes user behavior and content consumption patterns for BrightTV, a subscription-based streaming platform. The goal is to generate actionable insights that support the Customer Value Management (CVM) team in growing the subscription base.

 OBJECTIVES
Analyze user and usage trends
Identify key factors influencing content consumption
Recommend content strategies for low-consumption days
Propose initiatives to grow the user base

DATASET DESCRIPTION
The dataset includes:
User profiles (demographics, subscription info)
Viewer transactions (session-level consumption data)
All timestamps are provided in UTC and should be converted to South African Standard Time (SAST)
Each record represents one viewing session

KEY ANALYSIS AREAS
1. User & Usage Trends
Active users over time
Viewing frequency and session duration
Peak usage days and times

2. Consumption Drivers
Content type preferences (genres, formats)
Time-of-day and day-of-week patterns
User segmentation (e.g., heavy vs. casual users)

3. Content Strategy Recommendations
Identify low-consumption days
Recommend content types to boost engagement
Optimize release timing and scheduling

4. Growth Initiatives
Customer retention strategies
Personalized recommendations
Marketing and acquisition campaigns
Pricing or subscription model adjustments

RECOMMENDATIONS
Introduce targeted content releases on low-engagement days
Use personalized recommendations to increase watch time
Launch promotional campaigns to attract new users
Improve user segmentation for better CVM strategies
